
Configure and run a mouse Somatosensory Cortex microcircuit using preconfigured HBP model and data
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~



This Use Case allows a user to configure and launch a simulation of a detailed neuron model with a column shape on HBP HPAC infrastructure using `Unicore API <https://www.unicore.eu/>`__.

See Full Documentation :ref:`here <conf_run_hippo_pre>`.
